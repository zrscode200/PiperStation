# Work On

Orient to a registered project and route through the smallest useful Piper mode.

1. Read `AGENTS.md` and `STATION.md`.
2. Read the project ledger: `project.md`, `memory.md`, `decisions.md`, and `work/context-pack.md` when present.
3. Inspect the real repo path and current git state.
4. Classify scope and risk.
5. Choose direct work, Superpowers, Ralph, Review, or Finish Mode.

This command is prompt-driven routing, not a shell workflow engine.
